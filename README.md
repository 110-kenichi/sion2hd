# SION II HD for ossiloscope

1. Connect audio L,R ch to ossiloscope.
2. Set X-Y mode
3. Start www server on ./www folder and access to the URL by Web browser.
4. Set SIONII HDX OPTION "SOUND" to "OFF"
5. Start game

## NOTE

- xy-processor.js: X-Y mode renderer
- magic2.es6: MAGIC library

# SION II HD
HTML5 based X68k emulation environment specialized for running SION2 with high resolution graphics.

www/sion2 directory contains the original SION II. This is a part of OH10_5.XDF that is distributed at http://retropc.net/x68000/software/games/shooting/sion2/.
